Design System Approach

Aether Prism Style

This design system uses a modern tech approach with light theme featuring cutting-edge visual effects:

  • Animated Background : Subtle shifting radial gradients for depth
  • Glassmorphism : Frosted glass effects with backdrop blur on light backgrounds
  • Gradient Animations : Animated gradients on borders, text, and buttons
  • Micro-Interactions : Shimmer effects, ripple animations, and smooth transforms
  • Glow Effects : Soft glowing shadows for interactive elements
  • Tech Colors : Indigo, purple, and cyan palette on light backgrounds
  • Smooth Animations : Fluid transitions with cubic-bezier easing
  • 3D Effects : Subtle rotations and transforms on hover
